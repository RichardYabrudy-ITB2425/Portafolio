Proyecto 1

El Proyecto 1, SmartWaste, consistió en desarrollar un sistema automatizado para la gestión de residuos utilizando inteligencia artificial y visión por computadora. Configuré un modelo de IA en un Raspberry Pi para clasificar residuos, construí una cinta transportadora y optimicé la integración del hardware y software. También ayudé a crear un servidor web seguro para centralizar datos del sistema. Este proyecto combinó innovación tecnológica y sostenibilidad ambiental.

Proyecto 2

Colaboré con CellU Alot SL en un proyecto práctico para diseñar una sala de servidores (CPD) en su nueva sede para 150 empleados. Planifiqué el espacio para tres racks (comunicaciones, servidores y almacenamiento), teniendo en cuenta ventilación, seguridad y eficiencia. Además, elaboré un informe de medidas de protección y un presupuesto completo. Fue un ejercicio que combinó tecnología, estándares de seguridad y análisis económico.
